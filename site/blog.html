Журнал проекта

Интеграция с картографическим сервисом 2ГИС

10 мая 2025

Завершен важный этап проекта – предварительное размещение планов корпусов на картографической платформе 2ГИС. Это позволит в дальнейшем интегрировать нашу систему навигации с популярным картографическим сервисом.

Основные достижения этапа:

  • Определены точные геокоординаты корпусов
  • Согласованы форматы данных для интеграции
  • Подготовлена базовая структура для внедрения indoor-навигации

Создание 3D-моделей в Blender на основе планов SweetHome3D

17 апреля 2025

Успешно завершен процесс переноса архитектурных планов из SweetHome3D в Blender для создания детализированных 3D-моделей университетских помещений.

Выполненные работы:

  • Экспорт планировок из SweetHome3D в совместимом формате
  • Детальная проработка геометрии в Blender
  • Настройка материалов и текстур для реалистичной визуализации
  • Оптимизация моделей для веб-просмотра

Завершение работы над планами этажей

20 марта 2025

Закончен первый этап проекта – создание детальных планов этажей университета. Эта работа заложила фундамент для всех последующих этапов проекта.

Результаты работы:

  • Проведены замеры всех ключевых помещений
  • Созданы точные планы этажей с указанием всех помещений
  • Внесена информация о назначении каждого помещения
  • Отмечены все важные объекты инфраструктуры
